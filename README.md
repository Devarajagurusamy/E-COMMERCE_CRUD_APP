
## AI USAGE

## Copilot

Copilot was used as an AI pair programmer to speed up development. It helped generate repetitive code structures, component templates, API handlers, and TypeScript interfaces.

## ChatGPT

ChatGPT was used extensively throughout the project lifecycle for:

- Architecture and folder structure guidance
- MongoDB and Mongoose integration
- Authentication and JWT implementation
- Redux Toolkit state management
- Debugging runtime and API errors
- UI/UX improvements
- Code explanations and best practices

## CodeRabbit

CodeRabbit was used as an AI code reviewer to:

- Review code changes
- Suggest refactoring opportunities
- Detect potential bugs
- Improve code readability and maintainability
