# PHASE 6 - ADMIN DASHBOARD

Read **AGENT.md** first and follow it STRICTLY.

Follow **PHASE 0 - GLOBAL INSTRUCTIONS** before implementing anything.

Use:

* Authentication Module from Phase 2
* Product Module from Phase 3 & 4
* Cart Module from Phase 5

This phase is ONLY for the Admin Dashboard.

Do NOT implement:

* Orders
* Payment Gateway
* Analytics Charts
* Export CSV/PDF
* Notifications
* Real-time updates
* Socket.io

Implement Admin Dashboard only.

---

## Objective

Build a secure Admin Dashboard where only admin users can:

* View dashboard statistics
* View all products
* Search products
* Add products
* Edit products
* Delete products
* View registered users

---

# Access Control

ONLY:

```txt
role === "admin"
```

can access:

```txt
/ admin/*
```

If:

```txt
role !== "admin"
```

Return:

```txt
403 Forbidden
```

Or redirect:

```txt
/
```
